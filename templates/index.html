 Summary: Stock Forecast 

 Visualization 

The Purpose of this project is to analyze and predict stocks closing prices using different pediction algorithms and compare between them. We took historical stocks prices until a week ago from AlphaVantage. Then extracting latest stocks data and using Time series forecast. Facebook Prophet open source is used for this forecast.

Factors we looked at include: Date, Open, Close, High and Low prices. This site provides the visualizations created as part of the analysis which are deployed via Flask and Python.
