 Summary: Stock Forecast 

 Visualization 

The Purpose of thgis project is to understand analyzing stocks using Python Code. We took historical stocks prices. This will be done by extracting latest stocks data and using Time series forecast. Facebook open source is used for this forecast.

Factors we looked at include: Date, Open, Close, High and Low prices . This site provides the visualizations created as part of the analysis which are deployed via Flask and Python.
